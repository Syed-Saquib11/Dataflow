Configure Email Service
Step 1 of 4

Create a free EmailJS account

1. Click the button below to open emailjs.com
2. Click 'Sign Up' and create a free account
3. Free plan includes 200 emails/month
4. No credit card required

Connect your Email (Get Service ID)

1. In EmailJS dashboard click 'Email Services'
2. Click 'Add New Service'
3. Select Gmail (or any provider you use)
4. Click 'Connect Account' and sign in
5. Click 'Create Service'
6. Copy the Service ID shown (looks like: service_xxxxxxx)

Create Email Template (Get Template ID)

1. In EmailJS dashboard click 'Email Templates'
2. Click 'Create New Template'
3. Set these exact values:

Subject: Dataflow Password Reset Code
Body:
Hello Admin,

Your password reset code is: {{otp}}

This code expires at {{time}}.
Do not share this with anyone.

- Dataflow Teacher Portal

Note: Make sure to use {{to_email}} in the "To Email" field.

4. Click 'Save'
5. Copy the Template ID (looks like: template_xxxxxxx)

Get your Public & Private Keys

1. In EmailJS dashboard click your Account icon (top right corner)
2. Click 'Account'
3. Find 'Public Key' and 'Private Key' under API Keys section
4. Copy both keys
